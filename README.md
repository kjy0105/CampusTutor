# CampusTutor
https://testwebsite-theta-seven.vercel.app/#

대학생 튜터와 초·중·고 학생을 지역, 과목, 가능 시간대를 기준으로 연결하는 과외 매칭 웹 서비스입니다. 2026년 봄학기 인공지능 기초 프로그래밍 기말 프로젝트(Team 8)로 개발되었습니다.

## 프로젝트 개요

기존 과외 시장은 중개업체의 높은 수수료, 튜터 신뢰성 검증의 어려움, 그리고 매칭 이후 카카오톡과 전화로 흩어지는 소통 등의 문제를 안고 있습니다. CampusTutor는 과외 신청, 실시간 채팅, 후기 및 평점을 하나의 플랫폼에 통합하여 이 문제들을 해결하는 것을 목표로 합니다.

## 대상 사용자

- 학생(tutee): 자신의 지역과 학교, 희망 과목에 맞는 대학생 튜터를 찾으려는 초·중·고 학생
- 튜터(tutor): 본인의 전공과 가능 시간대에 맞춰 과외를 진행하려는 대학생
- 관리자(admin): 회원, 공지, 매칭, 후기, 재학 인증을 관리하는 운영진

## 주요 기능

- 회원가입 및 로그인: 학생과 튜터를 구분하여 가입하며, 지역과 과목, 가능 시간대 등 역할별 정보를 입력합니다.
- 지역 및 과목 기반 매칭: 시·도와 구·군, 과목, 30분 단위 주간 시간표의 겹침을 기준으로 추천 및 매칭이 이루어집니다.
- 양방향 매칭: 학생이 튜터에게 직접 신청하는 방식과, 학생이 공개 요청을 올리면 튜터가 시급과 소개를 담아 역제안을 보내는 방식을 모두 지원합니다.
- 1:1 실시간 채팅: 매칭된 상대와 실시간으로 대화하며 일정을 조율할 수 있고, 파일 첨부를 지원합니다.
- 알림 센터: 매칭, 채팅, 후기 등의 알림을 제공하며, 알림을 누르면 해당 채팅방이나 페이지로 바로 이동합니다.
- 후기 및 평점: 수업 후 학생이 별점과 후기를 남기고, 튜터는 알림을 받습니다.
- 재학 및 졸업 인증: 튜터가 증명서를 업로드하면 관리자가 검토 후 승인합니다.
- 관리자 콘솔: 통계 대시보드, 공지사항, 회원 관리, 건의사항 답변, 매칭 관리, 후기 관리, 재학 인증 승인을 제공합니다.

## 사용자 흐름

1. 학생이 회원가입을 하고 지역과 학교 정보를 입력한 뒤 로그인합니다.
2. 지역과 과목을 기준으로 추천된 튜터를 살펴보고 원하는 튜터에게 신청합니다.
3. 매칭된 튜터와 1:1 실시간 채팅으로 수업 일정을 조율합니다.
4. 수업이 끝나면 별점과 후기를 남기고, 튜터는 후기 알림을 받습니다.

## 기술 스택

- 프론트엔드: HTML, CSS, JavaScript (프레임워크 없는 단일 페이지 구조)
- 데이터베이스: Firebase Realtime Database (사용자, 튜터, 신청, 후기, 채팅, 알림의 실시간 동기화)
- 배포: Vercel (정적 사이트 호스팅, 글로벌 CDN)

## 파일 구성

- `index.html`: 홈, 대시보드, 마이페이지, 관리자 콘솔 화면과 각종 모달의 UI 구조를 정의합니다.
- `script.js`: Firebase 실시간 데이터 연동과 회원가입, 매칭, 채팅, 알림, 후기, 관리자 기능 등 모든 비즈니스 로직을 담고 있습니다.
- `style.css`: 전체 화면의 스타일을 정의합니다.
- `AI_USAGE.md`: 개발 과정에서 AI 도구를 어떻게 활용했는지 정리한 문서입니다.

## 참고 사항

코드에 포함된 Firebase 설정의 apiKey는 Firebase 웹 클라이언트 식별용 공개 값으로, 노출되어도 무방한 항목입니다. 데이터 접근 제어는 Firebase 콘솔의 보안 규칙(Database Rules)을 통해 관리합니다.
